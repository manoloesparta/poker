# Poker - Poquito Docker

Poker (Poquito Docker) is a bash implementation of Docker, making direct use of cgroups and namespaces for creating, managing, and interacting with containers.

# Requirements

* make
* curl
* jq
* shellcheck
* skopeo

# Features

- [ ] bocker pull
- [ ] bocker run
- [ ] bocker images
- [ ] bocker ps 
- [ ] bocker exec
- [ ] bocker rm
- [ ] bocker build
- [ ] bocker rmi
- [ ] networking
- [ ] resource limiting

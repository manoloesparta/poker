# bocker

docker implementation with Bash

# Requirements

* make
* curl
* jq
* shellcheck
* docker (used for downloading layers)

# Features

- [ ] bocker pull
- [ ] bocker run
- [ ] bocker images
- [ ] bocker ps 
- [ ] bocker exec
- [ ] bocker rm
- [ ] bocker build
- [ ] bocker rmi
- [ ] networking
- [ ] resource limiting
